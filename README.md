# 🌟 SkillSwap | Community Skill Exchange

SkillSwap is a premium full-stack web application designed for neighbors to trade skills using a time-based credit system. Users can offer expertise (e.g., AIML,FullStack,UI/UX) and "pay" mentors with virtual skill hours.

---

## ✨ Features

* **Secure Authentication**: Simple name-based login with profile creation.
* **Credit System**: Start with 5.0 hours; transfer credits to mentors for lessons.
* **Real-time Messaging**: Chat with neighbors to coordinate swaps.
* **Badges**: Automatically earn the "🌟 Top Mentor" badge when you receive your first credit.
* **Expert Search**: Filter users by name or specific skill.
* **Video Lessons**: Generate instant Jitsi meeting links for virtual learning.

---

## 🛠️ **Tech Stack**
* **Frontend**: HTML5, Tailwind CSS, Vanilla JavaScript.

* **Backend**: Node.js, Express.js.

* **Database**: MongoDB (Mongoose ODM).
